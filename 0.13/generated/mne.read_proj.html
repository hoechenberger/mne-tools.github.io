    mne.read_proj(fname)¶

    Read projections from a FIF file.

    Parameters:

    fname : string

    The name of file containing the projections vectors. It should end with -proj.fif or -proj.fif.gz.
    Returns:

    projs : list

    The list of projection vectors.
